# 銀行系統模擬

### 功能：

#### 1.建立帳戶
    -建立多個帳戶(引入CSV檔)
    -建立單一帳戶(stdin)

#### 2.刪除帳戶(需輸入密碼)
    -是否輸出(刪除帳號的)檔案資料

#### 3.檢視已有帳戶
    -檢視資料(不輸出檔案)
    -輸出檔案資料

#### 4.排序帳戶
    -依照名字
    -依照剩餘金額
    -依照開戶日期

#### 5.帳戶交易(需輸入密碼)
    -存款
    -提款
    -轉帳
    -投資(GAME)
        -DICE
        -GUESS
        -EVENT

#### 6.管理帳戶(需輸入密碼)
    -檢視帳戶個人信息
    -列出帳戶所有交易紀錄
    -修改個人信息
